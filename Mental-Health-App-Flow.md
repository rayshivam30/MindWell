## Quick Navigation Reference

### **Stage 1: Initial Entry**

The user lands on your app and sees three options: **Login**, **Create Account**, or **Continue as Guest**. Each path diverges here.

**Login Flow:** Email/Phone → Password → (Optional: Forgot Password Recovery) → Dashboard

**Sign Up Flow:** Personal Info → Password → User Type Selection → Email Verification → Profile Customization → Dashboard

### **Stage 2: Main Dashboard** (After Authentication)

Once logged in, users see a personalized welcome dashboard with:
- Daily mood check-in with emoji selector
- Quick access cards to 7 major features
- Today's summary (mood, sleep, stress)
- Upcoming appointments widget

### **Stage 3: Core Features** (All accessible from Dashboard)

**Mood Tracking**
- Select mood, intensity (1-10), triggers, associated feelings
- Saves entry → View AI-powered mood analytics and trends
- See patterns, most common triggers, and emotional insights

**Journaling**
- Write freely with rich text editor
- Optional: Add photos/audio attachments
- AI analyzes sentiment, detects themes, suggests coping strategies
- View previous entries and themes over time

**AI Therapy Chat**
- Real-time conversation with CBT-trained AI bot
- Suggests exercises, coping mechanisms, therapy booking
- End session anytime → View conversation summary

**Meditation & Mindfulness**
- Browse meditation library by category
- Select session (difficulty, duration, type)
- Play with haptic feedback, biometric tracking
- After session: Rate, view stats (heart rate, stress reduction)

**Video Therapy Sessions**
- Browse therapist profiles (specialties, rates, reviews)
- Book available time slots
- Before session: Join video call with one-click interface
- During session: Video, audio, chat enabled
- After session: Rate therapist, receive homework/notes

**Community Forum**
- Browse discussions by category (Anxiety, Sleep, Depression, etc.)
- Read posts and replies
- Create new discussions
- Like, comment, share within moderated space

**Wearable Integration**
- Connect Apple Watch, Fitbit, or other trackers
- View daily data: heart rate, sleep, steps, calories
- See stress analysis based on heart rate variability
- Get AI recommendations based on wearable patterns

**Profile & Settings**
- Edit personal information
- Update mental health concerns
- Manage notifications
- Privacy controls
- Two-factor authentication

### **Stage 4: Therapist Dashboard** (Separate Interface for Healthcare Providers)

Therapists see their own specialized interface:

**Therapist Home**
- Upcoming sessions with patients
- Unread messages
- New patient referrals
- Quick access to patient list, schedule, analytics

**Patient Management**
- View all patient profiles
- Access patient mood/journal data
- See treatment progress with graphs
- Add session notes and recommendations
- Track patient homework completion

**Scheduling**
- Calendar view of availability
- Book/reschedule appointments
- Set working hours
- Request time off

**Messaging**
- Communicate with patients
- Receive patient messages
- Send homework and resources
- Crisis alerts if detected

**Analytics & Billing**
- Track revenue and payments
- View patient progress statistics
- Generate tax reports
- Manage payout information
